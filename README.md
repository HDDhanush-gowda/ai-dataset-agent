# --- AI Dataset Agent---

Self-maintaining AI dataset agent for automated news collection, processing, and topic classification using NLP and Transformers.

---

#  --Features--
- Automated news data collection
- Data preprocessing & cleaning
- Topic classification system
- Transformer-based model support
- Flask web interface
- Historical dataset tracking
- CSV dataset generation

---

#  --AI / ML Used--
- NLP (Text Processing)
- Transformer Models (HuggingFace)
- Rule-based classification (fallback)

---

# -- Output Files--
- raw_data.csv
- processed_data.csv
- final_dataset.csv

---

#  --Tech Stack--
- Python
- Flask
- Pandas
- Transformers (BERT)
